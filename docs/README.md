# 미션 - 문자열 덧셈 계산기

## 📋 기능 구현 목록

### 덧셈 문자열 입력단계

- [x] 문자열 입력을 위한 문구 출력
- [ ] 계산을 위한 문자열 입력
  - [x] 쉼표(,) 또는 콜론(:)을 구분자로 가지는 문자열을 전달하는 경우 구분자를 기준으로 분리한 각 숫자의 합을 반환
  - [ ] 커스텀 구분자는 문자열 앞부분의 "//"와 "\n" 사이에 위치하는 문자를 커스텀 구분자로 사용
  - [ ] 빈문자열 입력할 경우 0반환

### 입력값 검증단계

- [ ] 계산대상이 숫자값이 아닐경우 예외검증
- [ ] 음수값일경우 예외검증

### 입력값 출력단계

- [ ] 분리된 모든값을 더한값 출력기능
