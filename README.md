## 1. 입출력
### 1.1. 입력 💋
- 숫자/구분자로 분리
- 커스텀 구분자 입력
- 예외처리
  - 기존 구분자 & 커스텀 구분자 외의 구분자가 입력될 경우
  - 커스텀 구분자를 문자열 맨 앞에 안 쓸 경우
### 1.2. 출력 💬
- "덧셈할 문자열을 입력해 주세요."
- "결과 : "

## 2. 계산 기능 🧮
- 구분자를 기준으로 숫자 저장
- 저장한 숫자를 모두 더하기
- 결괏값을 반환
