## 기능 명세서

### 1. 입력

- 문자열 입력

### 2. 커스텀 구분자를 판별

- 문자열 앞부분에서 커스텀 구분자를 확인
    - 커스텀 구분자 지정 문자가 "//"와 "\n" 가 아니면 예외 처리

### 3. 숫자 분리

- 구분자를 기준으로 숫자 분리
    - 구분자가 아닌 문자가 있을 경우 예외 처리
    - 구분자가 숫자 사이에 위치하지 않은 경우 예외 처리

### 4. 결과값 계산

- 모든 숫자들의 합산 구하기
    - 숫자가 양수가 아닐 경우 예외 처리

### 5. 출력

- 실행 결과대로 출력
- 입력값이 빈 문자열일 경우 결과값 0 출력
