# java-calculator-precourse

## 1. IO

**입출력에 관한 기능**

### input()

`camp.nextstep.edu.missionutils.Console`의 `readLine()`을 활용하여 입력 구현

- 구분자와 양수로 구성된 문자열

### output()

입출력 요구사항에 맞게 출력 구현

- 덧셈 결과

```
덧셈할 문자열을 입력해 주세요.
1,2:3
결과 : 6
```

## 3. Test

**예외 테스트 기능**

### 공통 구분자 예외
