# java-calculator-precourse

## 기능 목록
1. 입력한 문자열에서 숫자를 추출하여 더하는 '계산기' 구현
   
   1.1 문자열 입력받기
      1. 예외 처리 
         - '구분자와 양수' 외의 값이 들어온 경우
         - (추가) 구분자에서 양수가 들어온 경우
   
   1.2 문자열에서 숫자만 추출하기
      1. 구분자가 연속으로 오면 0으로 처리
      2. 빈문자열은 0으로 처리
      3. 예외 처리
         - (추가) 10억 넘는 양수가 들어온 경우
   
   1.3 더하기
      1. 예외 처리
         - (추가) 연산 시 10억이 넘어가는 경우
   
   1.4 출력 하기
   
   1.5 테스트 하기
