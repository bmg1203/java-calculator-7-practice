# [문자열 덧셈 계산기 ] 기능 목록

----

### 정의

쉼표(,) 또는 콜론(:) = 기본 구분자

### 입력

- 시작 문구("덧셈할 문자열을 입력해 주세요.") 출력 후 입력받음

### 출력

- 덧셈 결과 값을 출력

### 계산기

- 구분자(기본+커스텀)를 기준으로 분리
- 문자열 앞부분에 커스텀 구분자를 지정했는지 확인
- "//"와 "/n" 사이에 입력한 문자를 구분자에 추가
- 양수부터 입력했는지 확인
- 분리된 양수들의 합 계산

### 예외처리 `IllegalArgumentException`

- 구분자 또는 문자열로 입력했는지 확인
- 양수를 입력했는지 확인
- "//"와 "\n" 사이에 하나의 문자를 입력했는지 확인
