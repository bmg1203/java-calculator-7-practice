# java-calculator-precourse

### 문자열 덧셈 계산기

---

## 기능 요구 사항

- 실행시 사용자로부터 문자열을 입력받음- ✅
- 구분자를 기준으로 입력된 문자열 분리 - ✅
- 커스텀 구분자 입력 및 문자열 분리 - ✅
    - 문자열에서 구분자 생성부 제거 - ✅
- 분리된 문자열에서 숫자를 더함 - ✅
- 잘못된 값 입력시 IllegalArgumentException 발생 - ✅
    - 숫자가 아닐경우 예외 발생 - ✅
    - 음수가 들어올 경우 예외 발생 - ✅
    - 커스텀 구분자는 숫자로 시작할 수 없음. - ✅
