# 문자열 덧셈 계산기

## 기능 구현 목록

---

### 기본 구분자 처리
- [x] `,` 혹은 `:`를 구분자로 구분하기 


### 커스텀 구분자 처리
- [x] 커스텀 구분자를 사용해 구분하기
  - [x] 커스텀 구분자가 한 자리일 경우
  - [x] 커스텀 구분자가 여러 자리일 경우 
  
    
### 예외 처리
- [x] 양수가 아닌 값이 입력될 경우 예외 발생
  - [x] 0일 경우 예외 발생
  - [x] 음수일 경우 예외 발생
  - [x] 문자열이 입력되었을 때 예외 발생

### 계산 기능
- [x] 추출한 값을 더하기
  - [x] 소수를 포함한 계산 (ex 1.25 + 3)
  - [x] 정수끼리의 계산
  - [x] int값(-21억~21억)을 넘어갔을 때 계산

### 출력 기능
- [x] 계산값을 출력하기
  - [x] 정수끼리의 계산에서는 정수 출력
  - [x] 소수가 포함된 계산에서는 소수 출력
