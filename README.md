## 구현 기능 목록

### 1.CalculatorView
- 입력 받기
    - [V] ex) 덧셈할 문자열을 입력해 주세요.
- 결과 출력
    - [V] ex) 결과 : 6 
### 2. CalculatorParser
- 구분자를 기준으로 파싱
    - [V] 기본 구분자는 ","와 ":"
    - [V] 커스텀구분자 추가
        - 문자열 앞에 "//" + 커스텀구분자 + "\n"
    - [V] 숫자로 형변환
- 예외처리
    - 커스텀구분자
        - [V] 빈 값
        - [V] 숫자(표현식과 구분 불가)
    - 표현식
        - [V] 정수가 아닌 값
            - 정의되지않은 구분자 포함
        - [V] 빈 값
        - [V] 자바의 Integer 범위 초과 시
### 3. Calculator
- 각 숫자의 합을 계산해서 반환
    - [V] 주어진 리스트의 합 계산
- 예외처리
    - [V] 자바의 Integer 범위 초과 시
