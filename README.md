# java-calculator-precourse

### 요구 사항
- 입력
    - [x] 쉼표(,) 또는 콜론(:)을 구분자로 가지는 문자열을 입력받는다.
    - [x] 쉼표(,), 콜론(:), 커스텀 구분자, 양수 이외의 문자가 있으면 잘못된 입력이다.
    - [x] 입력이 빈 문자열이면 계산 결과는 0이다.
    - [x] 입력이 잘못된 경우, IllegalArgumentException를 발생시킨 후 종료한다.
    
- 계산
    - [x] 입력으로부터 받은 숫자 리스트에서 총합을 구한다.
- 출력
    - [x] 덧셈 결과를 출력한다.

### 실행 결과 예시

```
덧셈할 문자열을 입력해 주세요.
1,2:3
결과 : 6
```
