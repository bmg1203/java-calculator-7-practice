# java-calculator-precourse

# 구현 기능 목록
## validator

---
- 커스텀 구분자 파싱
- 구분자를 기준으로 숫자를 분리
  - 커스텀 구분자, 기본 구분자(쉼표, 콜론)가 아닌 구분자인 경우 에러 발생
  - 숫자가 아닌 다른 문자열이 있는 경우 에러 발생
  - 숫자가 양수가 아닌 경우 에러 발생


## util

---
- 문자열 입력 기능 구현
- 결과 출력 기능 구현
- 문자열을 문자 리스트로 변환


## controller

---
- 문자열 덧셈 계산기 구현
  - 에러가 발생하면 애플리케이션 종료
