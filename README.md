# java-calculator-precourse

# 기능 요구 사항
- [x] 문자열을 쉼표","와 콜론":"을 기준으로 분리하여 분리된 각 숫자의 합을 반환한다
- [x] 문자열 앞에 "//"와 "\n" 사이에 구분자를 입력하여 커스텀 구분자를 추가한다
- [x] 잘못된 문자열이 입력될 경우 IllegalArgumentException을 반환하고 프로그램을 종료한다

# 입력
구분자와 양수로 이루어진 문자열  

# 출력
구분자를 기준으로 분리한 양수를 모두 더한 값

# 실행 결과 예시
덧셈할 문자열을 입력해 주세요.  
1,2:3  
결과 : 6
