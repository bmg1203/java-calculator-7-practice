# java-calculator-precourse

### 기능 명세
1. 계산기 구현하기
  - [x] 사용자 입력 받기
    - [ ] 빈문자열 입력시 결과 0으로 처리하기
  - [x] 커스텀 구분자 설정 여부 확인
  - [x] 구분자를 기준으로 숫자 분리
  - [x] 분리된 숫자의 합 계산
  - [ ] 계산 결과 출력
  
2. 사용자의 잘못된 입력 예외처리하기
  - [ ] 잘못된 숫자 입력 (ex. 음수)
  - [ ] 커스텀 구분자, 쉼표, 콜론 외의 문자 입력
  - [ ] 잘못된 커스텀 구분자 입력 (ex. 문자가 아닌 문자열 입력)
