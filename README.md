# 문자열 계산기

## 1. Application.java
- 프로그램의 메인 시작점.
- `main()` 메서드에서 입력을 받아 계산기 실행.

## 2. StringCalculator.java
- 문자열을 받아 합계를 계산하는 기능 제공.
- 기본 구분자(쉼표, 콜론)와 커스텀 구분자 처리.
- 잘못된 입력에 대해 예외 처리.

## 3. InputParser.java
- 입력 문자열에서 구분자를 추출하고, 구분자와 숫자 분리.
- 커스텀 구분자(`//`와 `\n` 사이의 값) 처리.

## 4. TokenSplitter.java
- 주어진 구분자에 따라 문자열을 개별 숫자로 분리.
- 구분자에 따른 다중 문자열 처리.
