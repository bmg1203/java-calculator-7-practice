# java-calculator-precourse

# 문자열 덧셈 계산기

## 구현할 기능 목록

- [x] 사용자로부터 문자를 입력받는 기능
- [x] 쉼표(,) 또는 콜론(:)을 구분자로 가지는 문자열을 전달하는 경우 구분자를 기준으로 분리한 각 숫자의 합을 반환하는 기능
- [x] 커스텀 구분자를 지정하는 기능
- [x] 입력받은 숫자를 더하는 기능
- [x] 상수 목록 추가
- [x] 아무것도 입력하지 않았을 때 0을 출력하는 기능
- [x] 결과를 출력하는 기능

## 예외 사항 처리

- [x] 사용자가 숫자가 아닌 값을 입력하는 경우 예외 처리
- [x] 사용자가 음수를 입력하는 경우 예외 처리
