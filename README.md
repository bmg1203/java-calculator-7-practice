# java-calculator-recourse
## 기능 목록 단위
1. 빈 문자열 또는 null 입력 시 0을 반환하는 기능
2. 쉼표(,) 또는 콜론(:)을 구분자로 사용하는 문자열에서 숫자를 추출하여 합산하는 기능
    - ex:
        - "" => 0
        - "1,2" => 3
        - "1,2:3" => 6
3. 커스텀 구분자를 지원하는 기능
    - 문자열 형식: "//[구분자]\n[숫자들]"
    - ex:
        - "//;\n1;2;3" => 6
4. 잘못된 입력 값(음수 또는 비숫자) 시 IllegalArgumentException 발생시키는 기능
5. 모든 입력은 양수로만 제한하는 기능
