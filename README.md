# java-calculator-precourse

## 구현할 기능 목록

### 1. 핵심 기능
- [x] 문자열을 구분자로 나누기
- [x] 구분자로 나누어진 문자열들을 숫자로 변환하기
- [x] 변환된 숫자들의 합 계산하기

### 2. 추가 기능
- [x] 계산 결과 출력하기
- [x] 사용자로부터 문자열 입력받기
- [x] 커스텀 구분자를 지정하는지 검사하기
- [x] 구분자에 커스텀 구분자 추가하기

### 3. 예외 검사
- [ ] 입력된 정수가 양의 정수인지 검사하기
- [ ] 문자열에서 양의 정수도 구분자도 아닌 문자 검사하기
- [ ] 입력된 문자열에 구분자가 연속으로 있는지 검사하기
- [ ] 입력된 문자열 끝에 공백이 있는지 검사하기
- [ ] 커스텀 구분자가 문자 2개 이상인지 검사하기
- [ ] 커스텀 구분자가 숫자인지 검사하기
