## 구현 후

### 객체 정리

- Input 객체
    - 문자열을 입력 받을 책임

- Delimiters 객체
    - ~~구분자 반환할 책임~~   
      -> **구분자가 아닌 것을 문자열에 포함시켰을 경우 예외를 던지는** 책임
      (//\n 사이에 있는 구분자가 무엇인지 밝혀야 함)

- NumberExtractor 객체
    - 원시 문자열에서 숫자를 추출해 배열로 돌려줄 책임

- Adder 객체
    - 받은 숫자 List를 계산할 책임

- Output 객체
    - 받은 문자열(결과)을 문구와 함께 출력할 책임

- Calculator 객체
    - 흐름에 따라 모든 객체 생성하는 책임

## 구현 전 설계

목표: 입력한 문자열에서 숫자를 추출하여 더한 후 결과를 출력

### 필요 기능 작성

- 입력 받기
- 잘못된 값을 입력한 경우 예외 발생시키기
    - 아무 것도 입력하지 않았을 경우
    - 음수를 입력했을 경우
    - 구분자 이외의 문자를 입력한 경우
    - 구분자 사이에 숫자가 없을 경우
- 입력 받은 문자열에서 숫자 추출하기
    - 커스텀 구분자가 있는지 파악하기
- 추출한 숫자 더하기
- 결과 출력하기

---

### 유스케이스

유스케이스명: 입력한 문자열에서 숫자를 추출하여 더한다.

일차 액터: 사용자

주요 성공 시나리오:

1. 시스템은 “덧셈할 문자열을 입력해 주세요” 라는 문구를 출력한다.
2. 사용자가 구분자와 양수로 구성된 문자열을 입력해 덧셈을 요청한다.
3. 시스템은 숫자를 추출해 더한 후 사용자에게 제공한다.
