# java-calculator-precourse

## 기능 요구 사항
- 숫자를 더하는 계산기를 구현한다.
- 입력한 문자열에서 구분자를 기준으로 숫자를 분리한다.
- 커스텀 구분자를 지정할 수 있다.
- 사용자가 잘못된 값을 입력할 경우 `IllegalArgumentException`을 발생시킨 후 프로그램을 종료한다.

## 구현할 기능 목록
- [x] 숫자 클래스 작성
- [x] 덧셈 기능을 하는 계산기 클래스 작성
- [x] I/O 관련 View 작성
- [ ] 구분자를 기준으로 분리
- [ ] 분리한 값들이 양의 정수인지 확인
- [ ] 커스텀 구분자 판단하는 로직 작성
- [ ] 예외 처리
