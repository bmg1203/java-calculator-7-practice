## 패키지 구조

# 기능 목록
## 입력
- [ ] 입력 프롬프트를 출력한다.
- [ ] 한 줄의 문자열을 입력 받는다.

## 빈 입력 처리
- [ ] 입력이 비어 있을 경우 0을 반환한다.

## 숫자 추출
1. 입력이 숫자(0~9)로 시작하는 경우 </br>
   - [ ] 입력 문자열을 쉼표(,) 또는 콜론(:)으로 분리하여 숫자만 추출한다.
2. 입력이 '/'로 시작하는 경우 </br>
   - [ ] "//"와 "\n" 사이의 문자를 추출한다.
   - [ ] 입력 문자열을 추출한 문자로 분리하여 숫자만 추출한다.

## 덧셈
- [ ] 추출한 숫자들을 모두 더한다.

## 결과
- [ ] 출력 프롬프트와 함께 더한 값을 출력한다.
