# 🧮 문자열 덧셈 계산기

------

우아한테크코스 프리코스 1주차 미션


## 📌 개요

------

문자열로 숫자를 입력받고, 숫자를 더해서 출력한다


## 📋 기능 목록

------

- Model
    - [ ] 입력받은 문자열에서 숫자를 가져오는 기능
    - [ ] 골라낸 숫자를 더하는 기능


- View
    - [ ] 입력 안내메세지를 출력하는 기능
    - [x] 띄어쓰기 없이 문자열을 입력받는 기능
    - [ ] 덧셈을 실행한 결과를 출력하는 기능


- Controller
    - [ ] 계산기을 실행시키는 기능


- Exception
    - [ ] 문자열에 숫자가 없는 경우
    - [ ] int 데이터의 범위를 넘는 경우
    - [ ] 커스텀 구분자로 '\\'를 사용하는 경우


- Utils
    - [x] 숫자를 판단하는 기능
