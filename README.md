# 문자열 계산기

## ? 구현할 기능 목록

### 1?? 문자열 입력 및 기본 검증
- [ ] Console을 통해 사용자로부터 문자열 입력 받기
  ```java
  String input = Console.readLine();
  ```
- [ ] 입력값이 null이거나 빈 문자열인 경우 0을 반환
- [ ] 입력값 형식이 올바르지 않을 경우 `IllegalArgumentException` 발생

### 2?? 구분자 처리
- [ ] 기본 구분자(쉼표, 콜론) 처리
  - [ ] 쉼표(,)를 구분자로 사용하여 문자열 분리
  - [ ] 콜론(:)을 구분자로 사용하여 문자열 분리
  ```
  예시: "1,2:3" → [1, 2, 3]
  ```
- [ ] 커스텀 구분자 처리
  - [ ] "//"와 "\n" 사이의 문자를 커스텀 구분자로 인식
  - [ ] 커스텀 구분자를 이용하여 문자열 분리
  ```
  예시: "//;\n1;2;3" → [1, 2, 3]
  ```

### 3?? 문자열 파싱 및 변환
- [ ] 구분자로 분리된 문자열을 숫자로 변환
- [ ] 숫자가 아닌 값이 포함된 경우 예외 처리
- [ ] 음수가 포함된 경우 예외 처리

### 4?? 계산 및 결과 출력
- [ ] 분리된 숫자들의 합계 계산
- [ ] 계산 결과 출력

## ? 프로그램 실행 예시
```
덧셈할 문자열을 입력해 주세요.
1,2:3
결과 : 6
```

## ?? 제약 조건
* **JDK 버전**: 21
* **사용 가능한 라이브러리**:
  - `camp.nextstep.edu.missionutils.Console` 의 `readLine()`만 사용 가능
* **외부 라이브러리**: 사용 금지
* **System.exit()**: 사용 금지
* **코드 컨벤션**: Java 코드 컨벤션 준수

## ? 입력 형식
* 구분자와 양수로 구성된 문자열
* 커스텀 구분자: `//[구분자]\n` 형식으로 시작하는 문자열

## ? 출력 형식
* 모든 숫자의 덧셈 결과 값

## ?? 예외 처리
* 잘못된 값 입력 시 `IllegalArgumentException` 발생
* 프로그램 종료 처리

## Commit Message Format
```
<type>(scope): <subject> - Subject line
<BLANK LINE>               - 줄 바꿈으로 구분한다.
<body>                     - Message body
```
- Subject line
  - 변경 사항에 대한 간단한 설명을 한다.
  - 70자를 넘기지 않도록 한다.
- Message body
  - 수정 이유와 전후 비교 설명을 한다.
  - 명령형 현재 시제로 작성한다.
  - 70자를 넘기면 줄바꿈한다.
