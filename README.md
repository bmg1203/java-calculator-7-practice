- - -

### 구현 기능 목록

1. 쉼표(,)와 콜론(:)을 구분자로 숫자의 합을 계산하는 기능 구현

   > 예시 : &nbsp;&nbsp; "1,2:3" &nbsp;&nbsp; => &nbsp;&nbsp; 6

2. 커스텀 구분자를 통해 숫자의 합을 계산하는 기능 추가

   > 예시(커스텀 구분자 ';' 활용) : &nbsp;&nbsp; "//;\n1;2;3" &nbsp;&nbsp; => &nbsp;&nbsp; 6

3. 잘못된 값 입력에 대한 예외처리 기능 추가
