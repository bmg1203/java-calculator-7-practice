# java-calculator-precourse

## 기능 명세

1. 입력

- [ ] 입력 메시지를 출력한다.
- [ ] 입력을 받는다.

2. 파싱

- [ ] 커스텀 구분자가 존재하는지 확인한다. 존재한다면, 구분자 집합에 추가한다.
- [ ] 구분자 **(, ; 커스텀구분자**)가 있다면, 문자를 분리하여 배열에 저장한다.
- [ ] 문자 배열에 숫자가 아닌 문자가 있다면 IllegalArgumentException을 던진다.


3. 결과

- [ ] 분리한 각 숫자를 합한다.
- [ ] 결과물을 출력한다.

4. 고민

- 입력은 구분자와 양수로 구성된 문자열이다.
- 숫자의 범위는 어디까지 정해야 될지 생각해보자.
- 구분자가 숫자라면?
