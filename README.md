# 문자열 덧셈 계산기

## 개요
입력한 문자열에서 숫자를 추출하여 더하는 계산기입니다.

## 기능 목록
1. **문자열 입력받는 기능**
  - 구분자를 포함한 문자열을 입력받습니다.
  - 문자열은 양수와 구분자로 구성되어야 합니다.
    
2. **입력받은 문자열에서 숫자 저장하는 기능**
  - 입력받은 문자열에서 구분자를 기준으로 숫자를 분리하고 저장합니다.
    
3. **숫자 합을 계산하는 기능**
  - 저장된 숫자들의 합을 계산하여 반환합니다.
    
4. **출력하는 기능**
  - 계산된 합을 출력 요구 사항에 맞게 출력합니다.
