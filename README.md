# java-calculator-precourse

## 기능 목록

### 문자열 입력 기능 
  - 커스텀 구분자를 추가한다면 구분자를 따로 저장하고 해당 부분을 문자열에서 제거한다
    - 만약 구분자가 숫자일 경우 오류를 발생시킨다.

### 문자열 분리 기능
  - 기본+입력받은 구분자들을 기준으로 문자열을 분리한다
    - 숫자가 없을경우 오류를 발생시킨다.
    - 분리한 문자열에 숫자를 제외한 문자가 남아있을 경우 오류를 발생시킨다.
  - 분리한 숫자를 리스트에 저장한다.

### 덧셈 기능
- 입력받은 문자열을 전부 더한다. 

### 출력 기능
- 더한 결과를 반환받아 출력한다.
