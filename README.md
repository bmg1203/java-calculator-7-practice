## 문자열 덧셈 계산기
<br>

````
덧셈할 문자열을 입력해 주세요.
1,2:3
결과 : 6
````
###  기능 목록
 
- [x] 문자열 입력 
- [x] 문자열 토큰으로 나누기 (__","__ or __":"__)
- [x] __"//"__ __"\n"__ 입력시 사이에 문자를 구분자로 사용
  - ~~구분자 뒤에 공백이 있을 때 에러(편의성을 위해 에러가 아닌 허용)~~
- [ ] 잘못된 입력 값 예외처리
    - ~~양수가 아닌 경우~~
    - ~~숫자가 아닌 경우~~
    - 구분자가 다른 경우
    - 
- [x] 구분된 양수의 합 구하기
