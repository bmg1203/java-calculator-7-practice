## 구현 기능 목록
### 1. 문자열 입력 기능
### 2. 입력받은 문자열을 바탕으로 계산 기능
- #### 커스텀 구분자 확인
    - 입력받은 문자열이 빈 문자열일 경우 0 반환
    - "//"와 "\n" 문자열 사이의 구분자 확인
    - 두 문자열 중 하나가 없는 경우나 사이에 구분자가 없는 경우 예외처리
    - 커스텀 구분자가 없는 경우 기본 구분자(',', ':')를 구분자로 지정
- #### 구분자를 기준으로 숫자 분리 및 계산
    - 분리된 숫자의 값이 기대되는 값과 다른 경우 예외처리
### 3. 계산 결과 출력
