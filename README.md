# java-calculator-precourse

### Application

애플리케이션의 전반적인 동작을 담당한다.

- [ ] Calculator 인스턴스 생성
- [ ] View 입력 메시지 출력
- [ ] View 결과 메시지 출력

### View

콘솔을 이용해 사용자와의 상호작용을 담당한다. 

- [x] 문자열 입력 요구 메시지 출력
- [x] 사용자 입력
- [x] 결과 출력


### Calculator

문자열을 검증한다. 문자열 덧셈 계산을 담당한다.

- [x] 커스텀 구분자 확인
- [x] 숫자의 합 구하기
- [x] 예외 발생 상황
    - [x] 올바르지 않은 커스텀 구분자 정의   
    - [x] 정의되지 않은 구분자 사용
    - [x] 음수 비허용
