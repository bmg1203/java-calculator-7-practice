# java-calculator-precourse

## 기능 요구 사항 분석 후, 구현할 기능 목록 (v1 - 2024.10.17)

### 1. 입력값 해석
- [ ] 커스텀 구분자 사용 여부 확인
  
- [ ] 문자열 유효성 확인 및 구분자를 통한 숫자 추출
    - 추출한 숫자는 양수여야 함
    - 커스텀 구분자로 숫자가 들어올 경우, 해당 수는 구분자로 우선함
    - 추출 과정에서의 에러는 모두 IllegalArgumentException을 발생시키고 애플리케이션을 종료함

### 2. 계산기 관리
- [ ] 입력값 받기
    - 입력값은 주어진 라이브러리를 이용함
- [ ] 숫자 덧셈
