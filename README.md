## 계산기 기능 요구사항 정리

> 입력한 문자열에서 숫자를 추출하여 더하는 계산기를 구현한다. 

- [1] main 에서 문자열 입력 받기

- [2] 구분자 parsing
  - defalut 쉼포, 콜론 
  - //, \n 사이의 구분자
  - 추가 구분자는 존재 할 수 있고 존재 하지 않을 수 있다.
  - 구분자의 크기는 1이다. 

- [3] 숫자 parsing
  - 구분자가 연속되는 경우 0으로 취급한다.

- [4] 계산 값 반환
  - 파싱된 숫자를 모두 합하여 반환한다.
