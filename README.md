### 주요 기능
- [X] Default 구분자("**,**", "**:**")를 통한 숫자 파싱 기능
  - {숫자},{숫자}:{숫자}
- [X] Custom 구분자 파싱 기능
  - //{구분 문자열}\n{숫자}{구분 문자열}{숫자}
- [X] 입력에서 숫자 문자열 추출 기능
- [X] ~~구분자가 연속될때 숫자를 0으로 처리하는 기능~~
  - ex) {구분자1}**ᵛ**{구분자2}{숫자}{구분자3}
  - [X] **취소** 입력형식에 대한 에러로 수정
- [X] 숫자간 덧셈 기능
