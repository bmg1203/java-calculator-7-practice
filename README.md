# java-calculator-precourse

## 구현할 기능 목록

- **계산기 기능**
    - 행동
        - [x] 커스텀 구분자를 지정한다.
        - [x] 문자열을 구분자를 기준으로 분리한다.
            - [x] 기본 구분자를 기준으로 분리한다.
            - [x] 커스텀 구분자를 기준으로 분리한다.
        - [x] 구분자를 기준으로 분리한 각 숫자를 합한다.
    - 상태
        - [x] 기본 구분자
            - 쉼표(,)와 콜론(;)이다.
        - [x] 커스텀 구분자
            - [x] 한 자만 가능하다.
            - [x] 숫자가 아닌 문자만 가능하다.
        - [x] 구분한 숫자 값
            - [x] 숫자만 가능하다.
            - [x] 양수만 가능하다.


- **입출력 기능**
    - 행동
        - [x] 입력 안내 문구를 출력한다.
        - [x] 사용자로부터 문자열을 입력받는다.
        - [x] 각 숫자를 합한 결과를 출력한다.
