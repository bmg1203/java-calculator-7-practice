# java-calculator-precourse

## 기능구현 목록
1. 덧셈할 문자를 입력받는다.
2. 빈문자열 검사한다. 빈문자열인 경우 빈 List 반환한다.
3. 구분자 , : 사용했는지 검사한다.
    - [예외] , : 사용한 경우 List로 변환한다.
4. [예외] 커스텀 구분자 사용한 경우, List로 변환한다.
   - 숫자입력 부분의 빈 문자열 검사한다. 빈 문자열인 경우 빈 List 반환한다.
   - [예외] 변환된 List 숫자들이  모두 양수인지 검사한다.
6. List 숫자들을 모두 더한다.
7. 더한 값을 출력한다.
