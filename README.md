# 1주차 프리코스 : 문자열 덧셈 계산기

## 🛠️ 구현 기능 목록

- 입력 메세지 출력
- 사용자에게 문자열 입력 받기
- 입력 받은 문자열 확인
    - 커스텀 구분자를 지정했는지 확인
        - 예외 처리
        - 구분자를 "//" 와 "\n"이 감싸지 않는 경우
        - 구분자가 하나의 문자가 아닌경우
        - 구분자가 숫자인 경우
    - 문자열이 구분자와 양수로 이루어져 있는지 확인
        - 예외 처리
        - 숫자와 구분자 이외의 문자가 존재하는 경우
- 구분자를 기준으로 숫자 분리하기
- 분리한 숫자들을 더하여 합 출력
