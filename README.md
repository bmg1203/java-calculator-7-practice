# 문자열 덧셈 계산기
## InputView
- [ ] `덧셈할 문자열을 입력해주세요.` 를 출력한다.
- [ ] 사용자로부터 문자열을 입력받는다.
  - [ ] 공백이 아닌 문자열을 입력했는지?
  - [ ] 커스텀 구분자를 사용했는지?
    - [ ] 커스텀 구분자로 문자1개를 사용했는지?
  - [ ] 기본 구분자를 옳바르게 사용했는지? 
  - [ ] 구분자를 제외한 문자열들이 숫자로 변환이 가능한지?
  - [ ] 숫자가 양수인지?
## OutputView
- [ ] 문자열을 기본 구분자와 커스텀 구분자로 분리 및 정수 변환 후 모두 더하기 연산한다.
- [ ] `결과 : ` 와 함께 계산 결과를 출력한다.
