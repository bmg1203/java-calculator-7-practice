# java-calculator-precourse

1. InputHanler / OutputHandler 구조 잡기

2. InputValid 체크하기
   - 문자열에서 ,와 ; 구분자 제외하고 숫자가 맞는지 확인
   - 양수로 구성된 문자열인지 확인
   - 구분자가 연속으로 입력되었는지 확인
   - 문자열의 마지막 인덱스의 문자가 양수인지 확인

3. 커스텀 구분자 조건 확인
   - 문자열의 앞부분이 "//"와 "\n" 사이에 문자가 있는 구조로 되어있는지 확인
   ex) "//;\n1;2;3"  // result = 6

4. 더하기 계산 함수 생성
   - 기본 구분자 방식과 커스텀 구분자 방식으로 나누어 생성

5. 테스트 진행
