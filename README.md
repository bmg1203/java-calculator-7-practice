# java-calculator-precourse

## 기능 목록
- [ ] 유저의 문자열 입력 기능
  - [ ] 입력값이 , 또는 : 구분자인지 검증
  - [ ] 입력값이 커스텀 구분자인지 검증
  - [ ] 추출한 입력 값이 숫자인지 검증 
- [ ] 추출한 숫자 덧셈 기능

## 체크리스트
- [ ] JDK 21 버전에서 실행 가능하는가
- [ ] 사용자가 입력하는 값은 camp.nextstep.edu.missionutils.Console의 readLine()를 사용하였는가
- [ ] 사용자가 잘못된 값을 입력할 경우 `IllegalArgumentException`을 발생시킨 후 애플리케이션은 종료되는가
