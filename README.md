# java-calculator-precourse
## 기능 목록
- [ ] 사용자 입력을 받는다
- [ ] 입력을 검증한다
  - 공백만 입력했나?
  - 입력 형식에 맞지 않는가?
- [ ] 어떤 구분자인지 판단한다
  - 기본 구분자인가?
  - 커스텀 구분자인가?
- [ ] 입력을 구분자로 분리한다
- [x] 문자가 양수인지 검증한다
  - 음수인지 확인한다
  - 숫자 외 문자가 존재하는지 확인한다
  - 길이(1자리 ~ 15자리)를 확인한다
- [x] 양수들로 변환한다
  - 기본값 0은 포함하지 않는다
- [ ] 덧셈을 계산한다
- [ ] 덧셈 결과를 검증한다
  - 15자리 수 이하인가?
- [ ] 덧셈 결과를 출력한다
