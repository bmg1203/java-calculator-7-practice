# java-calculator-precourse

## 기능 목록

1. 문자열 입력받기
2. 구분자 기준으로 문자열 분리
    - 문자열이 "//"로 시작할 경우, "\n" 이전의 문자를 커스텀 구분자로 추출
    - 기본 구분자(쉼표(,), 콜론(:)) 또는 커스텀 구분자로 문자열을 분리하여 숫자를 추출
3. 숫자의 합 계산
    - 추출된 숫자들의 합을 계산하여 반환
    - 빈 문자열일 경우 0을 반환
4. 예외 처리
    - 잘못된 값 입력될 경우 `IllegalArgumentException`을 발생 시킨 후 프로그램 종료
        - 구분자 이외의 숫자가 아닌 문자가 포함된 경우
        - 음수가 입력된 경우
        - 커스텀 구분자가 비어있을 경우
        - 숫자가 범위를 초과한 경우
