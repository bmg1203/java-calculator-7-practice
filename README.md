# java-calculator-precourse
## 기능 요구 사항
- [x] 구분자(쉼표 또는 콜론) 기준으로 숫자 분리
- [x] 문자열을 숫자 리스트로 변환
- [x] 모든 수의 합 계산
- [x] 커스텀 구분자 지정(//와 \n 사이에 위치)
- [x] 사용자에게 양수로 구성된 문자열 입력받는 기능
- [ ] 커스텀 구분자 지정 문자 제거
- [x] 구분자 잘못 입력한 경우 예외 발생
- [x] 음수 입력한 경우 예외 발생
- [ ] 문자 입력한 경우 예외 발생

## 프로그래밍 요구 사항
- [ ] JDK 21 사용
- [ ] 자바 코드 컨벤션을 지키면서 프로그래밍
- [ ] 제공된 라이브러리 이외의 외부 라이브러리는 사용 불가능
